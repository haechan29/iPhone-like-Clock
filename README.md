### 기능 설명
- 뷰를 스크롤하면 뷰의 ``기울기``와 ``투명도``가 조절됩니다.<br/>
- 화면 중앙을 기준으로 Snap됩니다.<br/><br/>

### 사용 방법
단순히 코드를 **복사/붙여넣기**하는 것만으로 사용할 수 있습니다.<br/>

```
🧡 뷰의 구조가 단순하고, 함수의 역할이 명확하기 때문에 코드를 이해하기 쉽습니다.
```

  IPhoneLikeClock은 ``오전/오후``, ``시간``, ``분``을 나타내는 세 개의 Clock으로 구성됩니다.<br/>
